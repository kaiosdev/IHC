# UI Kit e Guia de Estilos - Gestão Acadêmica UFAM

**Conceito Visual:**
A identidade visual foi desenvolvida baseando-se nas cores institucionais da Universidade Federal do Amazonas (PROPESP/PROEXT), garantindo familiaridade para o corpo docente. No entanto, aplicamos uma modernização focada em **alto contraste** e **legibilidade** para atender à Persona de Acessibilidade (Prof. Carlos).

---

## 1. Paleta de Cores (Baseada na Identidade UFAM)

As cores foram extraídas dos portais oficiais e ajustadas para cumprir os requisitos WCAG (Web Content Accessibility Guidelines).

### Cores Institucionais
| Cor | Código HEX | Referência Visual | Contraste (WebAIM) |
| :--- | :--- | :--- | :--- |
| **Verde UFAM (Primária)** | `#00663C` | Cabeçalho PROPESP/PROEXT | **Pass (AA/AAA)** c/ Branco |
| **Cinza de Fundo (Secundária)**| `#F5F5F5` | Detalhes do Logo UFAM | Usar com texto preto (`#000`) |
| **Vermelho Alerta** | `#D32F2F` | Erros e Pendências | **Pass (AA)** |

### Cores de Interface (UI)
* **Surface (Fundo):** `#F5F5F5` (Quase branco, para descanso visual).
* **Text Primary:** `#212121` (Cinza escuro, para máxima leitura).
* **Text Secondary:** `#616161` (Para rótulos menos importantes).

> **Justificativa de Acessibilidade:**
> O Verde oficial foi levemente escurecido para garantir que, quando usado como fundo de botões com texto branco, o contraste seja suficiente para usuários com baixa visão.

---

## 2. Tipografia (Modernização)
Diferente dos sites institucionais antigos, utilizaremos uma tipografia sem serifa para facilitar a leitura rápida em telas e dispositivos móveis.

* **Família:** **Roboto** (Google Fonts).
* **Peso:**
    * **Bold (700):** Títulos e Botões.
    * **Regular (400):** Textos longos e listas.
* **Tamanho Base:** 16px (Nenhum texto será menor que 14px).

---

## 3. Ícones
* **Biblioteca:** Material Symbols (Rounded).
* **Estilo:** Sóbrio, utilizando o Verde UFAM para ícones ativos e Cinza para inativos.
